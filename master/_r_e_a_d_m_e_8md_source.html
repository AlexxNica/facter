1 Native Facter
2 =============
3 
4 An implementation of facter functionality in C++11, providing:
5 
6 * a shared library which gather facts about the system
7 * an executable for standalone command line usage
8 * a ruby file to enable `require 'facter'`.
9 
10 Please see our [extensibility document](https://github.com/puppetlabs/facter/blob/master/Extensibility.md) to learn more
11 about extending native facter using custom and external facts.
12 
13 Build Requirements
14 ------------------
15 
16 * GCC 4.8+ or Clang 5.0+ (OSX)
17 * CMake >= 3.2.2
18 * Boost C++ Libraries >= 1.54
19 * yaml-cpp >= 0.5.1
20 
21 Currently the leatherman library is vendored as a git submodule. To ensure submodules are initialized, please run:
22 
23 **`git submodule update --init`**
24 
25 Optional Build Libraries
26 ------------------------
27 
28 * OpenSSL - enables SSH fingerprinting facts.
29 * libblkid (Linux only) - enables the partitions fact.
30 * libcurl >= 7.18.0 - enables facts that perform HTTP requests.

61 ### Setup on Windows
62 
63 [MinGW-w64](http://mingw-w64.sourceforge.net/) is used for full C++11 support, and [Chocolatey](https://chocolatey.org) can be used to install. You should have at least 2GB of memory for compilation.
64 
65 * install [CMake](https://chocolatey.org/packages/cmake)
66 * install [MinGW-w64](https://chocolatey.org/packages/mingw)
67 
68  choco install mingw --params "/threads:win32"
69 
70 For the remaining tasks, build commands can be executed in the shell from Start > MinGW-w64 project > Run Terminal
71 
72 * select an install location for dependencies, such as C:\\tools or cmake\\release\\ext; we'll refer to it as $install
73 
74 * build [Boost](http://sourceforge.net/projects/boost/files/latest/download)
75 
76  .\bootstrap mingw
77  .\b2 toolset=gcc --build-type=minimal install --prefix=$install --with-program_options --with-system --with-filesystem --with-date_time --with-thread --with-regex --with-log --with-locale --with-chrono boost.locale.iconv=off
78 
79 * build [yaml-cpp](https://code.google.com/p/yaml-cpp/downloads)
80 
81  cmake -G "MinGW Makefiles" -DCMAKE_PREFIX_PATH=$install -DCMAKE_INSTALL_PREFIX=$install .
82  mingw32-make install
83 
84 * build [libcurl](http://curl.haxx.se/download)
85 
86  mingw32-make mingw32
87  cp -r include\curl $install\include
88  cp -r lib\libcurl.a $install\lib
89

91 
92  choco install cmake 7zip.commandline -y
93  choco install mingw --params "/threads:win32" -y
94  $env:PATH = "C:\tools\mingw64\bin;$env:PATH"
95  $install = "C:\tools"

123 Pre-Build

188 Install
189 -------
190 
191 You can install facter into your system:
192 
193  $ cd release
194  $ make && sudo make install
195 
196 By default, facter will install files into `/usr/local/bin`, `/usr/local/lib`, and `/usr/local/include`.
197 If the project is configured with Ruby in the PATH, facter.rb will be installed to that Ruby's vendor dir.
198 
199 To install to a different location, set the install prefix:
200 
201  $ cd release
202  $ cmake -DCMAKE_INSTALL_PREFIX=~ ..
203  $ make clean install
204 
205 This would install facter into `~/bin`, `~/lib`, and `~/include`.
