Detailed Description

template<typename T>
struct facter::util::scoped_resource< T >

Simple class that is used for the RAII pattern.

Used to scope a resource. When it goes out of scope, a deleter function is called to delete the resource. This type can be moved but cannot be copied.

TThe type of resource being scoped.
